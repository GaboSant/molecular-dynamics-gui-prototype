# Molecular Dynamics GUI

Graphical user interface developed for a no-code molecular dynamics platform designed to simplify the setup and execution of molecular simulations for users without programming experience.

This repository showcases the user interface component of the project, including the design and implementation of windows, navigation flows, input forms, and interaction elements that allow users to configure simulations through a visual environment.

## Overview

Molecular dynamics simulations often require users to interact with programming libraries, configuration files, and command-line tools. The goal of this interface is to provide a more accessible workflow by allowing users to configure simulations through a graphical application.

The interface was developed as part of an academic project focused on reducing the technical barriers associated with computational chemistry and molecular dynamics.

## Features

* Multi-window desktop interface
* Simulation parameter configuration forms
* File selection and management components
* Navigation between different workflow stages
* User-friendly layout for non-technical users
* Integration-ready design for molecular simulation workflows

## Technologies
